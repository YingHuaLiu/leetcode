斐波那契数，通常用 F(n) 表示，形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1

给你 n ，请计算 F(n) 。



我的思路：

1. 暴力递归

```js
var fib = function (n) {
  if (n < 2) {
    return n;
  }
  return fib(n - 1) + fib(n - 2);
};
```

2. 滑动数组

```js
var fib = function (n) {
  let a = 0, b = 1, result = 0;
  for (let i = 0; i < n; i++) {
    a = b;
    b = result;
    result = a + b;
  }
  return result;
};
```
